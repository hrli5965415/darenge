什么是HTML

  • HTML的全名叫做Hyper Text Markup Language，是一种用来编写网页的语言。

  • 不像c++，java或是python这些编程语言， HTML 是一种标记语言，是用来告诉浏览器网页结构的语言。

  • 一份HTML文件是由一连串的元素所组成。

  • HTML只能用来定义网页的结构，他不能用来改变网页的样式要改变网页的样式，例如更改字体大小，背景颜色等，那么你就需要使用CSS。。

以下为一个间单的HTML元素：

      <p>hellow world</p>
