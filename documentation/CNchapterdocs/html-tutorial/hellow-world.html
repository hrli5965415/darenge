创建HTML文件

要写一份HTML文件，首先在电脑上创建一个 空白文件 编写HTML时，最好是利用Visual Studio Code这类型的工具，但是你仍然可以在电脑上直接创建HTML文件，例如使用记事本(Windows用户)或者TextEdit(Mac用户)来编写HTML代码。 ，并将其命名为 index.html 文件的名字不拘，但是结尾必须是.html ，接著在文件里面输入： 

      <p>hellow world</p>

然后保存这份文件。

接著在 浏览器 你可以直接将文件拖到浏览器的分页上，或者在文件上按下滑鼠右键，选择Chrome来打开。 上打开它。或者，利用VScode的 local development server 要使用local development server，你可以在VScode的Extensions区域安装Live Server的插件。 也可以。

你的电脑萤幕会呈现上面的结果。

  • 左边的<p>叫做开始标签，代表一个元素的开始。

  • 右边的</p>叫做结束标签，代表一个元素的结束。

  • 中间的Hellow World代表元素携带的内容。
