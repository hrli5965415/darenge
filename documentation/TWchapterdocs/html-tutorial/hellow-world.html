創建HTML文件

要寫一份HTML文件，首先在電腦上創建一個 空白文件 編寫HTML時，最好是利用Visual Studio Code這類型的工具，但是你仍然可以在電腦上直接創建HTML文件，例如使用記事本(Windows用戶)或者TextEdit(Mac用戶)來編寫HTML代碼。 ，並將其命名為 index.html 文件的名字不拘，但是結尾必須是.html ，接著在文件裡面輸入： 

      <p>hellow world</p>

然後保存這份文件。

接著在 瀏覽器 你可以直接將文件拖到瀏覽器的分頁上，或者在文件上按下滑鼠右鍵，選擇Chrome來打開。 上打開它。或者，利用VScode的 local development server 要使用local development server，你可以在VScode的Extensions區域安裝Live Server的插件。 也可以。

你的電腦螢幕會呈現上面的結果。

  • 左邊的<p>叫做開始標籤，代表一個元素的開始。

  • 右邊的</p>叫做結束標籤，代表一個元素的結束。

  • 中間的Hellow World代表元素攜帶的內容。
