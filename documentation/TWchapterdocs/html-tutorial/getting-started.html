什麼是HTML

  • HTML的全名叫做Hyper Text Markup Language，是一種用來編寫網頁的語言。

  • 不像c++，java或是python這些編程語言， HTML 是一種標記語言，是用來告訴瀏覽器網頁結構的語言。

  • 一份HTML文件是由一連串的元素所組成。

  • HTML只能用來定義網頁的結構，他不能用來改變網頁的樣式要改變網頁的樣式，例如更改字體大小，背景顏色等，那麼你就需要使用CSS。。

以下為一個間單的HTML元素：

      <p>hellow world</p>
